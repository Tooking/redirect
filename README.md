# Страница редиректа для Telegram

Сайт перенаправляет пользователя на заданную ссылку. Удобно отправлять ссылку ботом в Telegram: при выборе «Открыть во внешнем браузере» откроется ваш сайт, затем произойдёт редирект на нужную страницу.

## Как задать целевую ссылку

### 1. Через config.js (одна ссылка для всего сайта)

Откройте `config.js` и замените `https://example.com` на свой URL:

```javascript
window.REDIRECT_CONFIG = {
  defaultUrl: 'https://ваш-сайт.com/страница'
};
```

Тогда ссылка для бота будет просто: `https://ваш-домен.com/`

### 2. Через параметр в URL (разные ссылки)

Добавьте параметр `url`, `to` или `u` к адресу страницы:

- `https://ваш-домен.com/?url=https://целевой-сайт.com`
- `https://ваш-домен.com/?to=https://другая-страница.com`

Для бота можно подставлять нужный URL в параметр при каждой отправке.

## Поведение

- При открытии страницы через 1,5 сек выполняется редирект на целевую ссылку.
- Если редирект не сработал, показывается кнопка «Перейти по ссылке».
- Рекомендация: в Telegram при отправке ссылки пользователям подсказать выбрать «Открыть во внешнем браузере» для корректного отображения.
